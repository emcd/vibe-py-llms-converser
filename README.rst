Installation üì¶
===============================================================================

Method: Install Executable Script
-------------------------------------------------------------------------------

Install via the `uv <https://github.com/astral-sh/uv/blob/main/README.md>`_
``tool`` command:

::

    uv tool install vibe-llms-converser

or, run directly with `uvx
<https://github.com/astral-sh/uv/blob/main/README.md>`_:

::

    uvx --from vibe-llms-converser converser

Or, install via `pipx <https://pipx.pypa.io/stable/installation/>`_:

::

    pipx install vibe-llms-converser

Method: Install Python Package
-------------------------------------------------------------------------------

Install via `uv <https://github.com/astral-sh/uv/blob/main/README.md>`_ ``pip``
command:

::

    uv pip install vibe-llms-converser

Or, install via ``pip``:

::

    pip install vibe-llms-converser
